## Getting Started

### Prerequisites

- A modern web browser
- Python 3 (for local development server)

### Running Locally

1. **Clone or download** the project to your local machine
2. **Navigate** to the project directory
3. **Start a local server**:
   - Using Python: `python3 -m http.server 8000`
   - Using Node.js: `npx http-server`
   - Or use the VS Code task: `Terminal > Run Task > Serve Website`
4. **Open your browser** and visit `http://localhost:8000`

### VS Code Tasks

The project includes a VS Code task for easy development:
- **Serve Website**: Starts a local Python HTTP server on port 8000

## File Descriptions

### `index.html`
The main HTML file containing:
- Semantic HTML structure
- Navigation header with mobile menu
- Hero section with call-to-action
- About, Services, and Contact sections
- Responsive footer

### `styles.css`
Comprehensive CSS file featuring:
- CSS Grid and Flexbox layouts
- Smooth transitions and animations
- Mobile-first responsive design
- Modern color scheme and typography
- Cross-browser compatibility

### `script.js`
JavaScript functionality including:
- Mobile navigation toggle
- Smooth scrolling navigation
- Form validation and submission
- Scroll-based navbar effects
- Intersection Observer animations
- Notification system
